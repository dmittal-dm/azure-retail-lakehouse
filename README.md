# Azure Retail Lakehouse

End-to-end retail data engineering project implementing a medallion
architecture using Databricks, Delta Lake, Azure Data Factory,
ADLS Gen2, PySpark, SQL, and Power BI.

## Architecture

Retail CSV Files
→ ADLS Gen2
→ Azure Data Factory
→ Azure Databricks
→ Bronze
→ Silver
→ Gold
→ Power BI
